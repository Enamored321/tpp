## 📌 Про проєкт

Цей репозиторій містить лендінг-сторінку бакалаврської кваліфікаційної роботи, присвяченої дослідженню ефективності алгоритму **Intelligent Water Drops (IWD)** для вирішення задачі комівояжера (TSP).

Сайт є публічною презентацією роботи: тема, мета, основні завдання та очікувані результати дослідження.

🔗 **[Переглянути сайт](https://enamored321.github.io/tpp/)**

---

## 🔬 Тема дослідження

**Алгоритм Розумних крапель води (IWD)** — метаевристичний алгоритм з ройового інтелекту, натхненний природним процесом розмивання ґрунту водою. Досліджується його ефективність у порівнянні з мурашиним алгоритмом (ACO) для вирішення NP-складних оптимізаційних задач на графах.

### Ключові терміни
`IWD` · `TSP` · `Задача комівояжера` · `Swarm Intelligence` · `Метаевристика` · `Оптимізація`

## 🚀 Локальний запуск

```bash
# 1. Клонувати репозиторій
git clone https://github.com/Enamored321/tpp.git
cd tpp

# 2. Встановити залежності
npm install

# 3. Запустити dev-сервер
npm run dev
```

Сайт буде доступний за адресою: `http://localhost:5173`

### Інші команди

```bash
npm run build      # Продакшн збірка
npm run preview    # Перегляд збірки
npm run lint       # Перевірка CSS + HTML
npm run format     # Форматування коду (Prettier)
```
