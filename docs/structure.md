# Структура проєкту

Проєкт розроблено як сучасну веб-сторінку (лендінг) для представлення результатів бакалаврської роботи.

## Каталоги та файли

- `index.html`: Головна сторінка з семантичною розміткою та SEO-тегами.
- `css/style.css`: Основні стилі проєкту (Mobile-first, Grid/Flexbox).
- `img/`: Директорія для графічних активів (SVG логотипи та ілюстрації).
- `docs/`: Технічна документація проєкту.
- `sitemap.xml` / `robots.txt`: Файли для пошукової оптимізації.
- `package.json`: Конфігурація інструментів розробки (Vite, Linters).

## Технологічний стек

- **HTML5**: Семантичні теги, Schema.org (JSON-LD).
- **CSS3**: CSS Variables, CSS Grid, Flexbox.
- **Vite**: Потужний інструмент для локальної розробки та збірки проєкту.
- **Linters**: Stylelint для контролю якості стилів.
